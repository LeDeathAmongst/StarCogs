PersonalChannels Help

Assign and edit personal channels.

Bot needs manage_channels guild permissions to utilize this cog.

mychannel

Description: Control of personal channels.

Usage: ,mychannel

mychannel perms

Description: Give users permissions on how many users they can to add in their channel.

Run this command without the perms argument to clear the permission config.

Usage: ,mychannel perms

mychannel assign

Description: Assign a personal text channel to someone.

Usage: ,mychannel assign

mychannel remove

Description: Delete member's personal channel.

Usage: ,mychannel remove

mychannel position

Description: Edit the channel position for someone's personal channel.

Usage: ,mychannel position

mychannel category

Description: Configure the category every personal text channel should be under.

Usage: ,mychannel category

mychannel pin

Description: Pin a message in the personal channel.

You can also reply to the message with invoking the command.

Usage: ,mychannel pin

mychannel friends

Description: Add or remove friends from your channel.

<add_or_remove> should be either add to add or remove to remove friends.

Usage: ,mychannel friends

mychannel friends list

Description: List your added friends.

Usage: ,mychannel friends list

mychannel name

Description: Change name of personal channel.

You cant use blacklisted names.

Usage: ,mychannel name

mychannel list

Description: Assigned channels list.

Usage: ,mychannel list

mychannel topic

Description: Change the topic of personal channel.

You can't use blacklisted words.

Usage: ,mychannel topic

mychannel blacklist

Description: Manage blacklisted names.

Usage: ,mychannel blacklist

mychannel blacklist add

Description: Add channel name to blacklist.

Members will not be able to change name to blacklisted names.

Usage: ,mychannel blacklist add

mychannel blacklist remove

Description: Remove channel name from blacklist.

Usage: ,mychannel blacklist remove

mychannel blacklist list

Description: List of blacklisted channel names.

Usage: ,mychannel blacklist list

mychannel unpin

Description: Unpin a message from the personal channel.

You can also reply to the message with invoking the command.

Usage: ,mychannel unpin

mychannel unassign

Description: Unassign personal text channel from someone.

Usage: ,mychannel unassign

mychannel delete

Description: Delete your personal channel.

Usage: ,mychannel delete

mychannel create

Description: Create a personal channel and assign it to the user.

  • <perms>: give the user permissions on how many users they want to add in their channel.

Usage: ,mychannel create
