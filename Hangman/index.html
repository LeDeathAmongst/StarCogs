Hangman Help

Play hangman with the bot.

hangman

Description: Play hangman with the bot.

Usage: ,hangman

hangmanset

Description: Config options for hangman.

Usage: ,hangmanset

hangmanset edit

Description: Set if hangman messages should be one edited message or many individual messages.

Defaults to True. This value is server specific.

Usage: ,hangmanset edit

hangmanset wordlist

Description: Change the wordlist used.

Extra wordlists can be put in the data folder. Wordlists are a .txt file with every new line being a new word. This value is server specific.

Usage: ,hangmanset wordlist

hangmanset wordlist current

Description: Show the current wordlist.

Usage: ,hangmanset wordlist current

hangmanset wordlist list

Description: List available wordlists.

Usage: ,hangmanset wordlist list

hangmanset wordlist default

Description: Set the wordlist to the default list.

Usage: ,hangmanset wordlist default
