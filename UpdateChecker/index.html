UpdateChecker Help

Get notices or auto-update cogs when an update is available for it's repo

cogupdater

Description: Group command for controlling the update checker cog.

Usage: ,cogupdater

cogupdater channel

Description: Sets a channel for update messages to go to.

If argument is not supplied, it will be sent to the default notifications channel(s) specified in [p]set ownernotifications. By default, this goes to owner DMs.

Usage: ,cogupdater channel

cogupdater task

Description: View the status of the task (the one checking for updates).

Usage: ,cogupdater task

cogupdater task status

Description: Get the current status of the update task.

Usage: ,cogupdater task status

cogupdater task error

Description: Gets the latest error of the update task.

Usage: ,cogupdater task error

cogupdater settings

Description: See settings for the Update Checker cog.

Right now, this shows whether the bot updates cogs automatically and what channel logs are sent to.

Usage: ,cogupdater settings

cogupdater embed

Description: Toggles whether to use embeds or colorful codeblock messages when sending an update.

Usage: ,cogupdater embed

cogupdater auto

Description: Changes automatic cog updates to the opposite setting.

Usage: ,cogupdater auto

cogupdater list

Description: Whitelist/blacklist certain repositories from which to receive updates.

Usage: ,cogupdater list

cogupdater list blacklist

Description: Blacklist certain repos from which to receive updates.

Usage: ,cogupdater list blacklist

cogupdater list blacklist clear

Description: Removes all repos from the blacklist

Usage: ,cogupdater list blacklist clear

cogupdater list blacklist remove

Description: Remove repos from the blacklist

Usage: ,cogupdater list blacklist remove

cogupdater list blacklist add

Description: Add repos to the blacklist

Usage: ,cogupdater list blacklist add

cogupdater list whitelist

Description: Whitelist certain repos from which to receive updates.

Usage: ,cogupdater list whitelist

cogupdater list whitelist clear

Description: Removes all repos from the whitelist

Usage: ,cogupdater list whitelist clear

cogupdater list whitelist add

Description: Add repos to the whitelist

Usage: ,cogupdater list whitelist add

cogupdater list whitelist remove

Description: Remove repos from the whitelist

Usage: ,cogupdater list whitelist remove
