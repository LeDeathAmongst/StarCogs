GameTracker Help

Cog to track when users start or stop playing a specific game.

gametracker

Description: Base command for game tracking.

Usage: ,gametracker

gametracker ignore

Description: Opt-out from being tracked.

Usage: ,gametracker ignore

gametracker addgame

Description: Add a game to the tracking list.

Usage: ,gametracker addgame

gametracker removegame

Description: Remove a game from the tracking list.

Usage: ,gametracker removegame

gametracker setlogchannel

Description: Set the channel where game activity updates will be logged.

Usage: ,gametracker setlogchannel
