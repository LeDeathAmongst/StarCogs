DblTools Help

Tools for Top.gg API.

dblset

Description: Group commands for settings of DblTools cog.

Usage: ,dblset

dblset poststats

Description: Set if you want to send your bot stats (Guilds and shards count) to Top.gg API.

Usage: ,dblset poststats

dblset rolerewards

Description: Settings for role rewards.

Usage: ,dblset rolerewards

dblset rolerewards role

Description: Set the role that will be added to new users if they have upvoted for your bot.

Usage: ,dblset rolerewards role

dblset rolerewards reset

Description: Reset current role rewards setup.

Usage: ,dblset rolerewards reset

dblset dailyrewards

Description: Settings for daily rewards.

Usage: ,dblset dailyrewards

dblset dailyrewards amount

Description: Set the amount of currency that users will receive on daily rewards.

Usage: ,dblset dailyrewards amount

dblset dailyrewards toggle

Description: Set wether you want [p]daily command usable or not.

Usage: ,dblset dailyrewards toggle

dblset dailyrewards weekend

Description: Set weekend bonus.

Usage: ,dblset dailyrewards weekend

dblset dailyrewards weekendamount

Description: Set the amount of currency that users will receive on week-end bonus.

Usage: ,dblset dailyrewards weekendamount

topgginfo

Description: Show information of a chosen bot on Top.gg.

bot: Can be a mention or ID of a bot. If not provided will default to the used bot.

Usage: ,topgginfo

dblwidget

Description: Send the widget of a chosen bot on Top.gg.

bot: Can be a mention or ID of a bot. If not provided will default to the used bot.

Usage: ,dblwidget

listdblvotes

Description: Sends a list of the persons who voted for the bot this month.

Usage: ,listdblvotes

dailyreward

Description: Claim your daily reward.

Usage: ,dailyreward
