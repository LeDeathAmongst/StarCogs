Trigger Help

trigger

Description: Group command for triggers.

Usage: ,trigger

trigger list

Description: List all triggers.

Usage: ,trigger list

trigger toggle

Description: Toggle a trigger.

Usage: ,trigger toggle

trigger create

Description: Create a trigger.

Variables can be used within the responses. user: The user that triggered the trigger. channel: The channel the trigger was triggered in. message: The message that triggered the trigger. guild: The guild the trigger was triggered in. uses: The number of times the trigger has been used. trigger: The name of the trigger that was triggered.

Example: {user} has triggered the trigger {trigger} in {channel} {uses} times.

Usage: ,trigger create
