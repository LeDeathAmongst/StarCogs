GitHubCards Help

GitHub Cards

ghsearch

Description: Search for issues in GitHub repo.

Protip: You can also search issues via prefix#s <search_query>!

Usage: ,ghsearch

githubcards

Description: GitHubCards settings.

Usage: ,githubcards

githubcards add

Description: Add a new GitHub repository with the given prefix.

Format for adding a new GitHub repo is "Username/Repository"

Usage: ,githubcards add

githubcards instructions

Description: Learn on how to setup GHC

This will be the first time that someone will ACTUALLY read instructions

Usage: ,githubcards instructions

githubcards remove

Description: Remove a GitHub repository with its given prefix.

Usage: ,githubcards remove

githubcards list

Description: List all prefixes for GitHub Cards in this server.

Usage: ,githubcards list
