Admin Help

A collection of server administration utilities.

addrole

Description: Add a role to a user.

Use double quotes if the role contains spaces. If user is left blank it defaults to the author of the command.

Usage: ,addrole

removerole

Description: Remove a role from a user.

Use double quotes if the role contains spaces. If user is left blank it defaults to the author of the command.

Usage: ,removerole

editrole

Description: Edit role settings.

Usage: ,editrole

editrole colour

Description: Edit a role's colour.

Use double quotes if the role contains spaces. Colour must be in hexadecimal format. Online colour picker

Examples: [p]editrole colour "The Transistor" #ff0000 [p]editrole colour Test #ff9900

Usage: ,editrole colour

editrole name

Description: Edit a role's name.

Use double quotes if the role or the name contain spaces.

Example: [p]editrole name "The Transistor" Test

Usage: ,editrole name

announce

Description: Announce a message to all servers the bot is in.

Usage: ,announce

announce cancel

Description: Cancel a running announce.

Usage: ,announce cancel

announceset

Description: Change how announcements are sent in this guild.

Usage: ,announceset

announceset clearchannel

Description: Unsets the channel for announcements.

Usage: ,announceset clearchannel

announceset channel

Description: Change the channel where the bot will send announcements.

Usage: ,announceset channel

selfrole

Description: Add or remove a selfrole from yourself.

Server admins must have configured the role as user settable. NOTE: The role is case sensitive!

Usage: ,selfrole

selfrole list

Description: Lists all available selfroles.

Usage: ,selfrole list

selfrole add

Description: Add a selfrole to yourself.

Server admins must have configured the role as user settable. NOTE: The role is case sensitive!

Usage: ,selfrole add

selfrole remove

Description: Remove a selfrole from yourself.

Server admins must have configured the role as user settable. NOTE: The role is case sensitive!

Usage: ,selfrole remove

selfroleset

Description: Manage selfroles.

Usage: ,selfroleset

selfroleset clear

Description: Clear the list of available selfroles for this server.

Usage: ,selfroleset clear

selfroleset add

Description: Add a role, or a selection of roles, to the list of available selfroles.

NOTE: The role is case sensitive!

Usage: ,selfroleset add

selfroleset remove

Description: Remove a role, or a selection of roles, from the list of available selfroles.

NOTE: The role is case sensitive!

Usage: ,selfroleset remove
