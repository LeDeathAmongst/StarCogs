CCRole Help

Custom commands
Creates commands used to display text and adjust roles

ccrole

Description: Custom commands management with roles

Highly customizable custom commands with role management.

Usage: ,ccrole

ccrole list

Description: Shows custom commands list

Usage: ,ccrole list

ccrole details

Description: Provide details about passed custom command

Usage: ,ccrole details

ccrole add

Description: Adds a custom command with roles

When adding text, put arguments in {} to eval them Options: {author}, {target}, {server}, {channel}, {message}

Usage: ,ccrole add

ccrole delete

Description: Deletes a custom command

Example: [p]ccrole delete yourcommand

Usage: ,ccrole delete
