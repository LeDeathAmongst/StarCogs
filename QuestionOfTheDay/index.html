QuestionOfTheDay Help

qotd

Description: Base for all question of the day commands.

Usage: ,qotd

qotd list

Description: Show questions in the main queue.

Usage: ,qotd list

qotd suggestions

Description: View all questions in the suggestion queue.

Usage: ,qotd suggestions

qotd posthere

Description: Set the current channel as where QOTDs should be posted.

Usage: ,qotd posthere

qotd approve

Description: Approve a suggestion using its id (see qotd suggestions).

This adds the suggestion to the main queue.

Usage: ,qotd approve

qotd remove

Description: Remove a question from the queue using its id (see qotd list).

Usage: ,qotd remove

qotd toggle

Description: Turn on or off automatic posting of questions of the day in this server.

Usage: ,qotd toggle

qotd deny

Description: Decline a suggestion and remove it from the suggestion queue.

For the suggestion's id, see qotd suggestions.

Usage: ,qotd deny

qotd post

Description: Post a question immediately.

A question will still be sent at the scheduled time if automatic posting is enabled.

Usage: ,qotd post

qotd suggest

Description: Add a question to the suggestion queue (it can be approved or denied by moderators).

Usage: ,qotd suggest

qotd add

Description: Add a question directly to the main queue (requires elevated permissions).

Usage: ,qotd add

qotd postat

Description: Set the time to post a QOTD every day in this server.

Usage: ,qotd postat
