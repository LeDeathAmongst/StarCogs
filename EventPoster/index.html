EventPoster Help

Create admin approved events/announcements

event

Description: All event related commands.

Usage: ,event

event show

Description: Show current event being run by a member

Usage: ,event show

event clear

Description: Delete a stored event so you can create more

[clear] yes/no to clear your current running event.

Usage: ,event clear

event ping

Description: Ping all the registered users for your event including optional message

[include_maybe=True] either true or false to include people who registered as maybe. [message] Optional message to include with the ping.

Usage: ,event ping

event leave

Description: Leave an event being hosted

Usage: ,event leave

event edit

Description: Edit various things in events

Usage: ,event edit

event edit title

Description: Edit the title of your event

<new_description> The new description for your event

Usage: ,event edit title

event edit remaining

Description: Show how long until your event will be automatically ended if available.

Usage: ,event edit remaining

event edit mayberemove

Description: Remove members from your events maybe list

[members...] The members you want to remove from your event

Usage: ,event edit mayberemove

event edit slots

Description: Edit the number of slots available for your event

<new_slots> The number of available slots for your events activity

Usage: ,event edit slots

event edit memberremove

Description: Remove members from your event (hopefully not against their will)

[members...] The members you want to add to your event

Usage: ,event edit memberremove

event edit maybeadd

Description: Add members to your events maybe list

[new_members...] The members you want to add to your event

Usage: ,event edit maybeadd

event edit memberadd

Description: Add members to your event (hopefully not against their will)

[new_members...] The members you want to add to your event

Usage: ,event edit memberadd

event join

Description: Join an event being hosted

Usage: ,event join

event set

Description: Manage server specific settings for events

Usage: ,event set

event set settings

Description: Show the current event settings.

Usage: ,event set settings

event set bypass

Description: Set whether or not admin approval is required for events to be posted.

<true_or_false> True or False whether or not to allow events to bypass admin approval.

Usage: ,event set bypass

event set viewlargelinks

Description: Show custom images available for events in this server

Usage: ,event set viewlargelinks

event set cleanup

Description: Set the events cleanup interval.

[time] How long events should be allowed to live before being automatically ended.

Note: If there is a timestamp for the event, the cleanup interval will check since the timestamp. If not it will check time after the event has been posted. Timestamp can be seen from the events embed.

Usage: ,event set cleanup

event set channel

Description: Set the Announcement channel for events

[channel] The channel events will be sent to. Providing no input will clear the channel.

If no channel is set events cannot be created.

Usage: ,event set channel

event set ping

Description: Set the ping to use when an event is announced

[everyone=False] True or False, whether to include everyone ping. [here=False] True or False, whether to include here ping. [role...] Is the role(s) you want to add to the list of pinged roles when an event is created.

If you want to ping here but not everyone you would do something like: - [p]event set ping false true

If you just want to set a few roles you can do: - [p]event set ping @role1 @role2

Usage: ,event set ping

event set thread

Description: Set whether or not to turn the announcement message into a thread for people to join and discuss in.

<true_or_false> True or False whether or not to allow events to bypass admin approval.

Usage: ,event set thread

event set remove

Description: Remove and end a current event.

<hoster> The member who is hosting the event.

Usage: ,event set remove

event set addplayerclass

Description: Add a playerclass choice for users to pick from on this server.

[emoji] Can be any emoji and is used on the drop down selector to help distinguish the classes. <player_class> The name of the player class you want to have as a server option.

Note: There is a maximum of 25 classes you can add. The class name can also only be a maximum of 100 characters.

Usage: ,event set addplayerclass

event set approvalchannel

Description: Set the admin approval channel

[channel] The channel you have restricted to people you trust to approve events. If no channel is provided this will be reset.

Note: This is required unless bypass has been enabled.

Usage: ,event set approvalchannel

event set roles

Description: Set the roles that are allowed to create events

[roles...] the role(s) that are allowed to create events. If not provided, there will be no restriction on who can create an event.

Usage: ,event set roles

event set removeplayerclass

Description: Remove a playerclass choice for users to pick from on this server.

<player_class> The name of the playerclass you want to remove.

Usage: ,event set removeplayerclass

event set links

Description: Set the custom thumbnail for events

<keyword> is the word that will be searched for in event titles. <link> needs to be an image link to be used for the thumbnail when the keyword is found in the event title.

Usage: ,event set links

event set largelinks

Description: Set the custom embed image for events

<keyword> is the word that will be searched for in event titles. <link> needs to be an image link to be used for the thumbnail when the keyword is found in the event title.

Usage: ,event set largelinks

event set class

Description: Set's the users default player class. If nothing is provided this will be rest.

[player_class] Your desired playerclass for events. This is listed next to your name when you register for an event. If this is changed during an event you have signed up for if the event updates with new members or changes in any way the event will reflect this change.

Usage: ,event set class

event set maxevents

Description: Set the maximum number of events the server can host.

[number_of_events] The maximum number of events this server can have running at one time.

Note: If this is set then the event author must cancel the event manually by either reacting to the x on the event itself or [p]clearevent. This can also be handled automatically with [p]eventset cleanup where events will last until the designated time after an event has started. Alternatively a mod or admin can cancel an event through [p]eventset remove

Usage: ,event set maxevents

event set defaultmax

Description: Set's the servers default maximum slots

[max_slots] The maximum number of slots allowed by default for events.

Usage: ,event set defaultmax

event set listplayerclass

Description: List the playerclass choices in this server.

Usage: ,event set listplayerclass

event set viewlinks

Description: Show custom thumbnails available for events in this server

Usage: ,event set viewlinks

event make

Description: Create an event

[members...] Add members already in the event you want to host. [max_slots=None] Specify maximum number of Slots the event can have, default is no limit. <description> provide a description for the event you're hosting. With custom keyword links setup this will add an image to the events thumbnail after being approved by an admin.

If a date or time is provided the timestamp in the event will try to display the correct time for everyone. For example [p]event Deep Stone Crypt Sunday at 9PM MDT will convert the "sunday at 9PM MDT" into a converted timestamp for everyone removing the need to know what MDT is in their own time. This also works for times relative to now, e.g. [p]event Last Wish in 3 hours will add the timestamp display in 3 hours from the time this message is posted.

Note: If a timezone is provided it must be the correct timezone according to daylight savings time. For example PST time may sometimes be UTC+8 in which case PDT must be used instead.

Usage: ,event make
