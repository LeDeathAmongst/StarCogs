ImageLog Help

Logs deleted images for moderation purposes.

imagelog

Description: View the current image log channel.

Usage: ,imagelog

imagelog log_moderator_self_deletes

Description: If disabled, users with Manage Message permission that delete their own image won't be logged. Enabled by default. True or False.

Usage: ,imagelog log_moderator_self_deletes

imagelog setchannel

Description: Sets the image log channel to the current channel.

Usage: ,imagelog setchannel
