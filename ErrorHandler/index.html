ErrorHandler Help

Adds ability to replace the output of the bots error handler when CommandInvokeError is raised, all other errors get handled by the old handler.

errorhandler

Description: Set and view the code for when a CommandInvokeError is raised.

Usage: ,errorhandler

errorhandler test

Description: This command contains an AssertionError purposely so that you can make sure your handler works

Usage: ,errorhandler test

errorhandler set

Description: Set the string to evaluate, use a python code block.

Environment variables: cf - redbot.core.utils.chat_formatting module ctx - context of invokation error - the error that was raised discord - discord.py library

Examples: - https://sourceb.in/Xalopv0AhZ

Usage: ,errorhandler set

errorhandler view

Description: View the string set to eval when a CommandInvokeError is raised.

Usage: ,errorhandler view
