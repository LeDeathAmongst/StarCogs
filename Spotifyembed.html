Spotifyembed Help

Automatically send a reply to Spotify links with a link to the embed preview. Convenient for mobile users who can finally listen to music samples from Discord, without needing an account.

setspotifyembed

Description: Set Spotify Embed settings

Automatically send a reply to Spotify links with a link to the embed preview. Convenient for mobile users who can finally listen to music samples from Discord, without needing an account.

Usage: ,setspotifyembed

setspotifyembed customurl

Description: Set a custom URL. The parsed original Spotify link will be added on at the end.

Example:

Custom URL: https://playsoju.netlify.app/?s= Final URL: https://playsoju.netlify.app/?s=https://open.spotify.com/...

Type "0" to reset to default. Type "1" to disable this feature. (show no text)

Usage: ,setspotifyembed customurl

setspotifyembed deleteoriginal

Description: Delete the original message after it's processed

Only for messages processed via auto-responding feature

Usage: ,setspotifyembed deleteoriginal

setspotifyembed note

Description: Change the text that appears before auto-responses

Type "0" to reset to default. Type "1" to disable this feature. (show no text)

Usage: ,setspotifyembed note

setspotifyembed enable

Description: Enable auto-responding to Spotify links

Usage: ,setspotifyembed enable

setspotifyembed disable

Description: Disable auto-responding to Spotify links

Usage: ,setspotifyembed disable

spotifyembed

Description: Return a Spotify embed link

Can set asMyself to true/false, for sending as webhook

Admins: To edit auto-reply and other settings, use [p]setspotifyembed

Usage: ,spotifyembed
