DiscordSearch Help

A cog to edit roles!

discordsearch

Description: Search for a message on Discord in a channel.

Warning: The bot uses the api for each search. Arguments: --author @user1 --author user2#1234 --author 0123456789 --mention @user1 --mention user2#1234 --mention 0123456789 --before now --after "25/12/2000 00h00" --pinned true --content "AAA3A-cogs" --regex "\[p\]" --contain link --contain embed --contain file --limit 100 (It's the limit of the number of messages taken into account in the search, not the number of results.)

Usage: ,discordsearch
