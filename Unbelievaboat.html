Unbelievaboat Help

Unbelievaboat Commands.

unbset

Description: Manage various settings for Unbelievaboat.

Usage: ,unbset

unbset add-reply

Description: Add a custom reply for working or crime.

Put {amount} in place of where you want the amount earned to be.

Usage: ,unbset add-reply

unbset fine-rate

Description: Set the min or max fine rate for crimes.

Usage: ,unbset fine-rate

unbset failure-rate

Description: Set the failure rate for crimes and robbing.

Usage: ,unbset failure-rate

unbset wallet

Description: Wallet Settings.

Usage: ,unbset wallet

unbset wallet max

Description: Set the max a wallet can have.

Usage: ,unbset wallet max

unbset wallet toggle

Description: Toggle the wallet system.

Usage: ,unbset wallet toggle

unbset list-replies

Description: List custom replies.

Usage: ,unbset list-replies

unbset default-replies

Description: Whether to use the default replies to work and crime.

Usage: ,unbset default-replies

unbset del-reply

Description: Delete a custom reply.

Usage: ,unbset del-reply

unbset cooldown

Description: Set the cooldown for the work, crime or rob commands. Minimum cooldown is 30 seconds.

The time can be formatted as so 1h30m etc. Valid times are hours, minutes and seconds.

Usage: ,unbset cooldown

unbset settings

Description: Current unbelievaboat settings.

Usage: ,unbset settings

unbset interest-rate

Description: Set the interest rate if unable to pay a fine from wallet.

Usage: ,unbset interest-rate

unbset betting

Description: Set the min or max betting amounts.

Usage: ,unbset betting

unbset payout

Description: Set the min or max payout for working or crimes.

Usage: ,unbset payout

cooldowns

Description: List your remaining cooldowns..

Usage: ,cooldowns

roulette

Description: Bet on the roulette wheel.

Current supported bets: Single - Any single number. Colors - Red/Black Halfs - 1st/2nd half Even Odd - Even or Odd Dozens - 1st/2nd/3rd Dozen (Groups of 12) Colums - 1st/2nd/3rd Column. - This is based on the English version of the roulette wheel.

Usage: ,roulette

roulette start

Description: Start a game of roulette.

Usage: ,roulette start

rouletteset

Description: Manage settings for roulette.

Usage: ,rouletteset

rouletteset settings

Description: Roulette Settings.

Usage: ,rouletteset settings

rouletteset payouts

Description: Set payouts for roulette winnings.

Note: payout is what your prize is multiplied by. Valid types: zero single color dozen odd_or_even halfs column

Usage: ,rouletteset payouts

rouletteset toggle

Description: Toggle roulette on and off.

Usage: ,rouletteset toggle

rouletteset time

Description: Set the time for roulette wheel to start spinning.

Usage: ,rouletteset time

wallet

Description: Wallet commands.

Usage: ,wallet

wallet balance

Description: Show the user's wallet balance.

Defaults to yours.

Usage: ,wallet balance

wallet leaderboard

Description: Print the wallet leaderboard.

Usage: ,wallet leaderboard

wallet set

Description: Set a users wallet balance.

Usage: ,wallet set

deposit

Description: Deposit cash from your wallet to your bank.

Usage: ,deposit

withdraw

Description: Withdraw cash from your bank to your wallet.

Usage: ,withdraw

addmoneyrole

Description: Add money to the balance of all users within a role.

Valid arguements are 'banks' or 'wallet'.

Usage: ,addmoneyrole

removemoneyrole

Description: Remove money from the bank balance of all users within a role.

Valid arguements are 'banks' or 'wallet'.

Usage: ,removemoneyrole

work

Description: Work for some cash.

Usage: ,work

crime

Description: Commit a crime, more risk but higher payout.

Usage: ,crime

rob

Description: Rob another user.

Usage: ,rob
