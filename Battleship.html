Battleship Help

Play battleship with one other person.

battleship

Description: Start a game of battleship.

Usage: ,battleship

battleshipstop

Description: Stop the game of battleship in this channel.

Usage: ,battleshipstop

battleshipboard

Description: View your current board from an ongoing game in your DMs.

Specify the channel ID of the channel the game is in.

Usage: ,battleshipboard

battleshipset

Description: Config options for battleship.

Usage: ,battleshipset

battleshipset extra

Description: Set if an extra shot should be given after a hit.

Defaults to True. This value is server specific.

Usage: ,battleshipset extra

battleshipset mention

Description: Set if players should be mentioned when their turn begins.

Defaults to False. This value is server specific.

Usage: ,battleshipset mention

battleshipset thread

Description: Set if a thread should be created per-game to contain game messages.

Defaults to False. This value is server specific.

Usage: ,battleshipset thread

battleshipset imgboard

Description: Set if the board should be displayed using an image.

Defaults to True. This value is server specific.

Usage: ,battleshipset imgboard
