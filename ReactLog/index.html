ReactLog Help

Log when reactions are added or removed.

reactlog

Description: Reaction logging configuration commands.

Usage: ,reactlog

reactlog blacklist

Description: Add/remove a member from reactlog blacklist.

Usage: ,reactlog blacklist

reactlog blacklist remove

Description: Remove a member from reactlog blacklist.

Usage: ,reactlog blacklist remove

reactlog blacklist add

Description: Add a member to reactlog blacklist.

Usage: ,reactlog blacklist add

reactlog grouped

Description: Set whether to group reaction logging embeds.

This is useful if your bot is in many servers and has a lot of users.

Usage: ,reactlog grouped

reactlog reactdel

Description: Enable/disable logging when reactions removed.

Usage: ,reactlog reactdel

reactlog channel

Description: Set the reactions logging channel.

Usage: ,reactlog channel

reactlog reactadd

Description: Enable/disable logging when reactions added.

Usage: ,reactlog reactadd

reactlog logall

Description: Set whether to log all reactions or not.

If enabled, all reactions will be logged. If disabled, only first added or last removed reactions will be logged.

Just a gentle reminder, it would be spammy if enabled.

Usage: ,reactlog logall

reactlog ignore

Description: Add/remove a channel from reactlog ignore list.

Usage: ,reactlog ignore

reactlog ignore add

Description: Add a channel to reactlog ignore list.

Usage: ,reactlog ignore add

reactlog ignore remove

Description: Remove a channel from reactlog ignore list.

Usage: ,reactlog ignore remove

reactlog settings

Description: Show current reaction log settings.

Usage: ,reactlog settings
