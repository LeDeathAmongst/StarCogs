Voteout Help

vote

Description: Start a voteout against a user.

Usage: ,vote

votesettings

Description: Change the settings for voteout.

Usage: ,votesettings

votesettings anonymousvotes

Description: Change whether or not votes are anonymous.

Usage: ,votesettings anonymousvotes

votesettings action

Description: Change the action to take on a voted out user.

Usage: ,votesettings action

votesettings mutuallyexclusiveroles

Description: Change the mutually exclusive roles for voteout.

This command overwrites the previously set roles.

These roles are roles that are allowed to voteout other users. They also impose a restriction that users with one of these roles cannot voteout a user with another one of these roles.

Usage: ,votesettings mutuallyexclusiveroles

votesettings button

Description: Change the button settings for voteout.

Usage: ,votesettings button

votesettings button style

Description: Change the style of the button.

1 is blurple, 2 is grey, 3 is green and 4 is red.

Usage: ,votesettings button style

votesettings button label

Description: Change the label of the button.

Variables: "{target}" will be replaced with the user's name that's being voted out. "{votes}" will be replaced with the number of votes. "{threshold}" will be replaced with the threshold. "{action}" will be replaced with the action to take on user if voteout succeeds.

Usage: ,votesettings button label

votesettings button emoji

Description: Change the emoji of the button.

Usage: ,votesettings button emoji

votesettings ignorehierarchy

Description: Change whether or not to ignore role hierarchy when users vote out other users.

Usage: ,votesettings ignorehierarchy

votesettings timeout

Description: Change the timeout for voteout.

Timeout can not be greater than 15 minutes. This accepts second and minute units, by default, it assumes a number to be seconds. e.g. 5 is 5 seconds, 5m is 5 minutes.

Usage: ,votesettings timeout

votesettings threshold

Description: Change the threshold for voteout.

This is the number of votes required to take action on a voted out user.

Usage: ,votesettings threshold
