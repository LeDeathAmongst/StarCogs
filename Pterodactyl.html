Pterodactyl Help

Pterodactyl allows you to manage your Pterodactyl Panel from Discord.

pterodactyl

Description: No description provided.

Usage: ,pterodactyl

pterodactyl command

Description: No description provided.

Usage: ,pterodactyl command

pterodactyl players

Description: No description provided.

Usage: ,pterodactyl players

pterodactyl power

Description: No description provided.

Usage: ,pterodactyl power

pterodactyl

Description: Pterodactyl allows you to manage your Pterodactyl Panel from Discord.

Usage: ,pterodactyl

pterodactyl players

Description: Retrieve a list of players on the server.

Usage: ,pterodactyl players

pterodactyl power

Description: Send power actions to the server.

Usage: ,pterodactyl power

pterodactyl power start

Description: Start the server.

Usage: ,pterodactyl power start

pterodactyl power stop

Description: Stop the server.

Usage: ,pterodactyl power stop

pterodactyl power restart

Description: Restart the server.

Usage: ,pterodactyl power restart

pterodactyl power kill

Description: Kill the server.

Usage: ,pterodactyl power kill

pterodactyl config

Description: Configure Pterodactyl settings.

Usage: ,pterodactyl config

pterodactyl config view

Description: View the current configuration.

Usage: ,pterodactyl config view

pterodactyl config messages

Description: Configure message settings.

Usage: ,pterodactyl config messages

pterodactyl config messages startup

Description: Set the message that will be sent when the server starts.

Usage: ,pterodactyl config messages startup

pterodactyl config messages shutdown

Description: Set the message that will be sent when the server stops.

Usage: ,pterodactyl config messages shutdown

pterodactyl config messages join

Description: Set the message that will be sent when a user joins the server. This is only shown in embeds.

Usage: ,pterodactyl config messages join

pterodactyl config messages leave

Description: Set the message that will be sent when a user leaves the server. This is only shown in embeds.

Usage: ,pterodactyl config messages leave

pterodactyl config regex

Description: Set regex patterns.

Usage: ,pterodactyl config regex

pterodactyl config regex blacklist

Description: Blacklist regex patterns.

Usage: ,pterodactyl config regex blacklist

pterodactyl config regex blacklist add

Description: Add a regex pattern to the blacklist.

Usage: ,pterodactyl config regex blacklist add

pterodactyl config regex blacklist remove

Description: Remove a regex pattern from the blacklist.

Usage: ,pterodactyl config regex blacklist remove

pterodactyl config regex join

Description: Set the regex pattern to match join messages on the server.

See documentation for more information.

Usage: ,pterodactyl config regex join

pterodactyl config regex leave

Description: Set the regex pattern to match leave messages on the server.

See documentation for more information.

Usage: ,pterodactyl config regex leave

pterodactyl config regex chat

Description: Set the regex pattern to match chat messages on the server.

See documentation for more information.

Usage: ,pterodactyl config regex chat

pterodactyl config regex achievement

Description: Set the regex pattern to match achievement messages on the server.

See documentation for more information.

Usage: ,pterodactyl config regex achievement

pterodactyl config regex server

Description: Set the regex pattern to match server messages on the server.

See documentation for more information.

Usage: ,pterodactyl config regex server

pterodactyl config invite

Description: Set the invite link for your server.

Usage: ,pterodactyl config invite

pterodactyl config url

Description: Set the base URL of your Pterodactyl Panel.

Please include the protocol (http/https). Example: https://panel.example.com

Usage: ,pterodactyl config url

pterodactyl config topic

Description: Set the topic for the console and chat channels.

Usage: ,pterodactyl config topic

pterodactyl config topic port

Description: Set the port of your server.

Usage: ,pterodactyl config topic port

pterodactyl config topic host

Description: Set the hostname or IP address of your server.

Usage: ,pterodactyl config topic host

pterodactyl config topic text

Description: Set the text for the console and chat channels.

Available placeholders: - .$H (hostname) - .$O (port) Available for Minecraft servers: - .$I (ip) - .$M (max players) - .$P (players online) - .$V (version) - .$D (description / Message of the Day)

Usage: ,pterodactyl config topic text

pterodactyl config serverid

Description: Set the ID of your server.

Usage: ,pterodactyl config serverid

pterodactyl config ip

Description: Mask the IP addresses of users in console messages.

Usage: ,pterodactyl config ip

pterodactyl config chat

Description: Configure chat settings.

Usage: ,pterodactyl config chat

pterodactyl config chat channel

Description: Set the channel to send chat output to.

Usage: ,pterodactyl config chat channel

pterodactyl config chat command

Description: Set the command that will be used to send messages from Discord.

Required placeholders: .$U (username), .$M (message), .$C (color) See documentation for more information.

Usage: ,pterodactyl config chat command

pterodactyl config console

Description: Configure console settings.

Usage: ,pterodactyl config console

pterodactyl config console channel

Description: Set the channel to send console output to.

Usage: ,pterodactyl config console channel

pterodactyl config console commands

Description: Enable or disable console commands.

Usage: ,pterodactyl config console commands

pterodactyl config api

Description: Set the API endpoint for retrieving user avatars.

This is only used for retrieving user avatars for webhook messages. See PlayerDB for valid endpoints. Usually, you should leave this as default.

Usage: ,pterodactyl config api

pterodactyl command

Description: Send a command to the server console.

Usage: ,pterodactyl command
