    Cloudflare Help

    A Red-Discordbot cog to interact with the Cloudflare API.

    images

    Description: Cloudflare Images provides an end-to-end solution to build and maintain your image infrastructure from one API. Learn more at https://developers.cloudflare.com/images/

    Usage: ,images

    images upload

    Description: Upload an image to Cloudflare Images.

    Usage: ,images upload

    images stats

    Description: Fetch Cloudflare Images usage statistics.

    Usage: ,images stats

    images delete

    Description: Delete an image from Cloudflare Images by its ID.

    Usage: ,images delete

    images info

    Description: Get information about a specific image.

    Usage: ,images info

    images list

    Description: List available images.

    Usage: ,images list

    loadbalancing

    Description: Cloudflare Load Balancing distributes traffic across your servers, which reduces server strain and latency and improves the experience for end users. Learn more at https://developers.cloudflare.com/load-balancing/

    Usage: ,loadbalancing

    loadbalancing list

    Description: Get a list of load balancers for a specific zone.

    Usage: ,loadbalancing list

    loadbalancing patch

    Description: Update the settings of a specific load balancer.

    Usage: ,loadbalancing patch

    loadbalancing create

    Description: Create a new load balancer for a specific zone.

    Usage: ,loadbalancing create

    loadbalancing info

    Description: Get information about a specific load balancer by its ID.

    Usage: ,loadbalancing info

    loadbalancing delete

    Description: Delete a load balancer by its ID.

    Usage: ,loadbalancing delete

    dnssec

    Description: DNSSEC info

    Usage: ,dnssec

    dnssec status

    Description: Get the current DNSSEC status and config for a specific zone.

    Usage: ,dnssec status

    dnssec delete

    Description: Delete DNSSEC on the currently set Cloudflare zone

    Usage: ,dnssec delete

    keystore

    Description: Fetch keys in use for development purposes only

    Usage: ,keystore

    keystore apikey

    Description: Fetch the current Cloudflare API key

    Usage: ,keystore apikey

    keystore bearertoken

    Description: Fetch the current Cloudflare bearer token

    Usage: ,keystore bearertoken

    keystore accountid

    Description: Fetch the current Cloudflare account ID

    Usage: ,keystore accountid

    keystore email

    Description: Fetch the current Cloudflare email

    Usage: ,keystore email

    keystore zoneid

    Description: Fetch the current Cloudflare zone ID

    Usage: ,keystore zoneid

    botmanagement

    Description: Cloudflare bot solutions identify and mitigate automated traffic to protect your domain from bad bots. Learn more at https://developers.cloudflare.com/bots/

    Usage: ,botmanagement

    botmanagement get

    Description: Get the current bot management config from Cloudflare.

    Usage: ,botmanagement get

    botmanagement update

    Description: Update a specific bot management setting.

    Usage: ,botmanagement update

    zones

    Description: Cloudflare command group.

    Usage: ,zones

    zones get

    Description: Get the list of zones from Cloudflare.

    Usage: ,zones get

    intel

    Description: Cloudforce One packages the vital aspects of modern threat intelligence and operations to make organizations smarter, more responsive, and more secure. Learn more at https://www.cloudflare.com/application-services/products/cloudforceone/

    Usage: ,intel

    intel ip

    Description: Query intelligence on a public IP address.

    Usage: ,intel ip

    intel domainhistory

    Description: Fetch and display category and domain history.

    Usage: ,intel domainhistory

    intel whois

    Description: Query WHOIS information for a given domain.

    Usage: ,intel whois

    intel asn

    Description: Fetch and display ASN intelligence from Cloudflare.

    Usage: ,intel asn

    intel domain

    Description: Query Cloudflare API for domain intelligence.

    Usage: ,intel domain

    intel subnets

    Description: Fetch and display ASN subnets intelligence from Cloudflare.

    Usage: ,intel subnets

    urlscanner

    Description: With Cloudflare’s URL Scanner, you have the ability to investigate the details of a domain, IP, URL, or ASN. Cloudflare’s URL Scanner is available in the Security Center of the Cloudflare dashboard, Cloudflare Radar and the Cloudflare API.

    Learn more at https://developers.cloudflare.com/radar/investigate/url-scanner/

    Usage: ,urlscanner

    urlscanner create

    Description: Start a new scan for the provided URL.

    Usage: ,urlscanner create

    urlscanner search

    Description: Search for URL scans by date and webpage requests.

    Usage: ,urlscanner search

    urlscanner screenshot

    Description: Get the screenshot of a scan by its scan ID

    Usage: ,urlscanner screenshot

    urlscanner scan

    Description: Scan a URL using Cloudflare URL Scanner and return the verdict.

    Usage: ,urlscanner scan

    urlscanner results

    Description: Get the result of a URL scan by its ID.

    Usage: ,urlscanner results

    urlscanner har

    Description: Fetch the HAR of a scan by the scan ID

    Usage: ,urlscanner har

    Usage: ,emailrouting

    emailrouting enable

    Description: Enable Email Routing for the selected zone

    Usage: ,emailrouting enable

    emailrouting add

    Usage: ,emailrouting add

    emailrouting disable

    Description: Disable Email Routing for the selected zone

    Usage: ,emailrouting disable

    emailrouting remove

    Description: Remove a destination address from your Email Routing service.

    emailrouting records

    Usage: ,emailrouting records

    emailrouting settings

    emailrouting rules

    emailrouting rules remove

    emailrouting rules list

    Usage: ,emailrouting rules list

    emailrouting list

    Description: List current destination addresses

    Usage: ,emailrouting list

    hyperdrive

    Description: Hyperdrive is a service that accelerates queries you make to existing databases, making it faster to access your data from across the globe, irrespective of your users’ location. Learn more at https://developers.cloudflare.com/hyperdrive/

    Usage: ,hyperdrive

    hyperdrive create

    Description: Create a new Hyperdrive

    Usage: ,hyperdrive create

    hyperdrive update

    Description: Update and return the specified Hyperdrive configuration.

    Usage: ,hyperdrive update

    hyperdrive delete

    Description: Delete a Hyperdrive.

    Usage: ,hyperdrive delete

    hyperdrive info

    Description: Fetch information about a specified Hyperdrive by its ID.

    Usage: ,hyperdrive info

    hyperdrive list

    Description: List current Hyperdrives in the specified account

    Usage: ,hyperdrive list

    hyperdrive patch

    Description: Patch a specified Hyperdrive by its ID with provided changes.

    Usage: ,hyperdrive patch

    r2

    Description: Cloudflare R2 Storage allows developers to store large amounts of unstructured data without the costly egress bandwidth fees associated with typical cloud storage services.

    Learn more at https://developers.cloudflare.com/r2/

    Usage: ,r2

    r2 info

    Description: Get info about an R2 bucket

    Usage: ,r2 info

    r2 stash

    Description: Upload a file to the specified R2 bucket

    Usage: ,r2 stash

    r2 create

    Description: Create a new R2 bucket

    Valid location hints

    apac - Asia-Pacific eeur - Eastern Europe enam - Eastern North America weur - Western Europe wnam - Western North America

    Usage: ,r2 create

    r2 recycle

    Description: Delete a file by name from an R2 bucket

    Usage: ,r2 recycle

    r2 delete

    Description: Delete a specified R2 bucket

    Usage: ,r2 delete

    r2 fetch

    Description: Fetch a file from an R2 bucket

    Usage: ,r2 fetch
