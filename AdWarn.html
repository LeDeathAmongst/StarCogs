AdWarn Help

adwarn

Description: Warn a user and send an embed to the default warning channel.

Usage: ,adwarn

removewarn

Description: Remove a specific warning from a user.

Usage: ,removewarn

warncount

Description: Get the total number of warnings a user has.

Usage: ,warncount

clearwarns

Description: Clear all warnings for a user.

Usage: ,clearwarns

unadwarn

Description: Clear the most recent warning for a user.

Usage: ,unadwarn

warnset

Description: Settings for the warning system.

Usage: ,warnset

warnset show

Description: Show the current warning channel.

Usage: ,warnset show

warnset channel

Description: Set the default channel for warnings.

Usage: ,warnset channel
