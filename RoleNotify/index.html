RoleNotify Help

Notify a user when they have a Role added or removed from them

More detailed docs: https://cogs.yamikaitou.dev/rolenotify.html

rolenotify

Description: Configure RoleNotify

Usage: ,rolenotify

rolenotify channel

Description: Set the channel to output Role Notifications to

Pass 0 to clear the channel Pass nothing to see the configured channel

Usage: ,rolenotify channel

rolenotify role

Description: Configure settings for a Role

Usage: ,rolenotify role

rolenotify role remove

Description: Set if the notification should be sent on Role Remove

should be any of these combinations, on/off, yes/no, 1/0, true/false

Usage: ,rolenotify role remove

rolenotify role message

Description: Set the notification message
