Alias Help

Create aliases for commands.

Aliases are alternative names/shortcuts for commands. They can act as both a lambda (storing arguments for repeated use) or as simply a shortcut to saying "x y z".

When run, aliases will accept any additional arguments and append them to the stored alias.

alias

Description: Manage command aliases.

Usage: ,alias

alias delete

Description: Delete an existing alias on this server.

Usage: ,alias delete

alias edit

Description: Edit an existing alias in this server.

Usage: ,alias edit

alias global

Description: Manage global aliases.

Usage: ,alias global

alias global edit

Description: Edit an existing global alias.

Usage: ,alias global edit

alias global list

Description: List the available global aliases on this bot.

Usage: ,alias global list

alias global delete

Description: Delete an existing global alias.

Usage: ,alias global delete

alias global add

Description: Add a global alias for a command.

Usage: ,alias global add

alias help

Description: Try to execute help for the base command of the alias.

Usage: ,alias help

alias list

Description: List the available aliases on this server.

Usage: ,alias list

alias show

Description: Show what command the alias executes.

Usage: ,alias show

alias add

Description: Add an alias for a command.

Usage: ,alias add
