DynamicChannelList Help

Create dynamically updating channel lists.

dynamicchannellist

Description: Group command for DynamicChannelList.

Usage: ,dynamicchannellist

dynamicchannellist color

Description: Set the color to use for embeds.

Usage: ,dynamicchannellist color

dynamicchannellist createauto

Description: Create an automatically updating channel list.

Usage: ,dynamicchannellist createauto

dynamicchannellist header

Description: Set the header for all embed messages.

Usage: ,dynamicchannellist header

dynamicchannellist removeauto

Description: Remove an automatically updating channel list.

This will not delete the message, only stop it from updating. message should be a link to the message.

Usage: ,dynamicchannellist removeauto

dynamicchannellist channelblacklist

Description: Toggle if a channel is blacklisted from appearing on channel lists.

If no channel is provided, the currently blacklisted categories will be listed.

Usage: ,dynamicchannellist channelblacklist

dynamicchannellist categoryblacklist

Description: Toggle if a category is blacklisted from appearing on channel lists.

If no category is provided, the currently blacklisted categories will be listed.

Usage: ,dynamicchannellist categoryblacklist

dynamicchannellist reloadauto

Description: Reload all automatically updating channel lists.

Usage: ,dynamicchannellist reloadauto

dynamicchannellist generate

Description: Generate a one-time channel list.

Usage: ,dynamicchannellist generate
