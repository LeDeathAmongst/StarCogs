GuildLog Help

Log when the bot joins or leaves a guild

guildlogset

Description: Configure GuildLog Settings

Usage: ,guildlogset

guildlogset embeds

Description: (Toggle) Embeds for join/leave log

Usage: ,guildlogset embeds

guildlogset view

Description: View GuildLog Settings

Usage: ,guildlogset view

guildlogset channel

Description: Set a channel for the bot to log guilds it leaves/joins

Usage: ,guildlogset channel

guildlogset leave

Description: Configure leave settings

Usage: ,guildlogset leave

guildlogset leave color

Description: Set the color of the guild leave embed

If embeds are on, this will be the leave color. Color value must be an integer

Usage: ,guildlogset leave color

guildlogset leave msg

Description: Set the guild leave message

Valid placeholders are: {guild} - the name of the guild the bot just joined {guildid} - the id of the guild the bot just joined {servers} - the amount of servers the bot is now in {botname} - the name of the bot {bots} - the amount of bots that were in the guild {members} - the member count of the guild

To set back to default just to default as the message value

Usage: ,guildlogset leave msg

guildlogset join

Description: Configure join settings

Usage: ,guildlogset join

guildlogset join color

Description: Set the color of the guild join embed

If embeds are on, this will be the join color. Color value must be an integer

Usage: ,guildlogset join color

guildlogset join msg

Description: Set the guild join message

Valid placeholders are: {guild} - the name of the guild the bot just joined {guildid} - the id of the guild the bot just joined {servers} - the amount of servers the bot is now in {botname} - the name of the bot {bots} - the amount of bots in the guild {members} - the member count of the guild

To set back to default just to default as the message value

Usage: ,guildlogset join msg
