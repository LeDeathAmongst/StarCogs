TheMovieDB Help

Search for informations of movies and TV shows from themoviedb.org.

tmdbset

Description: Setup TheMovieDB.

Usage: ,tmdbset

tmdbset version

Description: Shows the version of the cog

Usage: ,tmdbset version

tmdbset creds

Description: Set your TMDB API key

Usage: ,tmdbset creds

movie

Description: Search for a movie.

You can write the full name of the movie to get more accurate results.

Examples: - [p]movie the matrix - [p]movie the hunger games the ballad of songbirds and snakes

Arguments: - <query> - The movie you want to search for.

Usage: ,movie

tvshow

Description: Search for a TV show.

You can write the full name of the tv show to get more accurate results.

Examples: - [p]tvshow the simpsons - [p]tvshow family guy

Arguments: - <query> - The serie you want to search for.

Usage: ,tvshow

people

Description: Search for a person.

You can write the full name of the person to get more accurate results.

Examples: - [p]people tom hanks - [p]people meryl streep

Arguments: - <query> - The person you want to search for.

Usage: ,people
