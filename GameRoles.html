GameRoles Help

Grant roles when a user is playing a specific game.

gameroles

Description: Group command for game roles.

Usage: ,gameroles

gameroles delrole

Description: Stop a role from being managed by gameroles.

Roles with multiple words need to be surrounded in quotes. Accepts the ID of the role in case it was deleted.

Usage: ,gameroles delrole

gameroles recheck

Description: Force a recheck of your current activities.

Usage: ,gameroles recheck

gameroles addactivity

Description: Add an activity to trigger a role.

Roles and activities with multiple words need to be surrounded in quotes. You can get the name of your current activity with [p]gameroles currentactivity.

Usage: ,gameroles addactivity

gameroles addrole

Description: Sets a role to be managed by gameroles.

Roles with multiple words need to be surrounded in quotes. The bot's highest role needs to be above the role that you are adding and the bot needs permission to manage roles.

Usage: ,gameroles addrole

gameroles delactivity

Description: Remove an activity from triggering a role.

Roles and activities with multiple words need to be surrounded in quotes. You can get the name of your current activity with [p]gameroles currentactivity.

Usage: ,gameroles delactivity

gameroles listactivities

Description: List the activities that trigger a role.

Roles with multiple words need to be surrounded in quotes.

Usage: ,gameroles listactivities

gameroles currentactivity

Description: Get your current activity.

Usage: ,gameroles currentactivity

gameroles listroles

Description: List the roles currently managed by gameroles.

Usage: ,gameroles listroles

gameroleset

Description: Config options for gameroles.

Usage: ,gameroleset

gameroleset add

Description: Set if roles should be added when someone starts playing a game.

Defaults to True. This value is server specific.

Usage: ,gameroleset add

gameroleset remove

Description: Set if roles should be removed when someone stops playing a game.

Defaults to True. This value is server specific.

Usage: ,gameroleset remove
