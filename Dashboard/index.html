Dashboard Help

Interact with your bot through a web Dashboard!

Installation guide: https://red-web-dashboard.readthedocs.io/en/latest ⚠️ This package is a fork of Neuro Assassin's work, and isn't endorsed by the Org at all.

dashboard

Description: Get the link to the Dashboard.

Usage: ,dashboard

setdashboard

Description: Configure Dashboard.

Usage: ,setdashboard

setdashboard metaicon

Description: The website icon to use.

Default value: None Dev: <class 'str'>

Usage: ,setdashboard metaicon

setdashboard modalconfig

Description: Set all settings for the cog with a Discord Modal.

Usage: ,setdashboard modalconfig

setdashboard disabledthirdparties

Description: The third parties to disable.

Default value: [] Dev: Greedy[ThirdPartyConverter]

Usage: ,setdashboard disabledthirdparties

setdashboard defaultbackgroundtheme

Description: Set the default Background theme of the dashboard.

Default value: white Dev: typing.Literal['white', 'dark']

Usage: ,setdashboard defaultbackgroundtheme

setdashboard metatitle

Description: The website title to use.

Default value: None Dev: <class 'str'>

Usage: ,setdashboard metatitle

setdashboard metawebsitedescription

Description: The website short description to use.

Default value: None Dev: <class 'str'>

Usage: ,setdashboard metawebsitedescription

setdashboard metadescription

Description: The website long description to use.

Default value: None Dev: <class 'str'>

Usage: ,setdashboard metadescription

setdashboard showsettings

Description: Show all settings for the cog with defaults and values.

Usage: ,setdashboard showsettings

setdashboard secret

Description: Set the client secret needed for Discord OAuth.

Usage: ,setdashboard secret

setdashboard redirecturi

Description: The redirect uri to use for the Discord OAuth.

Default value: None Dev: <class 'dashboard.dashboard.RedirectURIConverter'>

Usage: ,setdashboard redirecturi

setdashboard defaultsidenavtheme

Description: Set the default Sidenav theme of the dashboard.

Default value: white Dev: typing.Literal['white', 'dark']

Usage: ,setdashboard defaultsidenavtheme

setdashboard supportserver

Description: Set the support server url of your bot.

Default value: None Dev: <class 'str'>

Usage: ,setdashboard supportserver

setdashboard resetsetting

Description: Reset a setting.

Usage: ,setdashboard resetsetting

setdashboard allowunsecurehttprequests

Description: Allow unsecure http requests. This is not recommended for production, but required if you can't set up a SSL certificate.

Default value: False Dev: <class 'bool'>

Usage: ,setdashboard allowunsecurehttprequests

setdashboard defaultcolor

Description: Set the default Color of the dashboard.

Default value: success Dev: typing.Literal['success', 'danger', 'primary', 'info', 'warning', 'dark']

Usage: ,setdashboard defaultcolor

setdashboard allinone

Description: Run the Dashboard in the bot process, without having to open another window. You have to install Red-Dashboard in your bot venv with Pip and reload the cog.

Default value: False Dev: <class 'bool'>

Usage: ,setdashboard allinone

setdashboard flaskflags

Description: The flags used to setting the webserver if all_in_one is enabled. They are the cli flags of reddash without --rpc-port.

Default value: [] Dev: Greedy[StrConverter]

Usage: ,setdashboard flaskflags
