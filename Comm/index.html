Comm Help

usercomm

Description: Manage usercomm connections.

Usage: ,usercomm

usercomm open

Description: Link the current DM to the Communications channel.

Usage: ,usercomm open

usercomm list

Description: List all users connected to the Communications channel alongside their IDs.

Usage: ,usercomm list

usercomm close

Description: Unlink the current DM from the Communications channel.

Usage: ,usercomm close

usercomm remove

Description: Forcibly remove a user from the Communications channel.

Usage: ,usercomm remove

usercomm addtrusted

Description: Add a trusted user who can forcibly remove users from the Communications channel.

Usage: ,usercomm addtrusted

usercomm removetrusted

Description: Remove a trusted user.

Usage: ,usercomm removetrusted
