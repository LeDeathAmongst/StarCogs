RoleManager Help

A cog to manage roles based on member statuses and activities.

rolemanager

Description: Group command for managing role assignments based on status and activities.

Usage: ,rolemanager

rolemanager status

Description: Check the current settings for role assignments.

Usage: ,rolemanager status

rolemanager setrolelistchannel

Description: Set the channel for the dynamic role list.

Usage: ,rolemanager setrolelistchannel

rolemanager setrole

Description: Set a role to be assigned for a specific member status or activity.

Usage: ,rolemanager setrole

rolemanager clearrole

Description: Clear the role assigned for a specific member status or activity.

Usage: ,rolemanager clearrole

rolemanager setlogchannel

Description: Set the channel for logging role assignments.

Usage: ,rolemanager setlogchannel
