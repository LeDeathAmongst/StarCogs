TextEditor Help

Edit and manipulate with text.

editor

Description: Base command for editing text.

Usage: ,editor

editor reverse

Description: Reverse the text.

Usage: ,editor reverse

editor multiply

Description: Multiply the text.

Usage: ,editor multiply

editor swapcase

Description: Swap the casing for text.

Usage: ,editor swapcase

editor trim

Description: Trim the outskirts of the text.

Usage: ,editor trim

editor formatnumber

Description: Format a number with commas.

Usage: ,editor formatnumber

editor wordcount

Description: Count the number of words appearing in the text.

Usage: ,editor wordcount

editor wrap

Description: Wrap the text.

Usage: ,editor wrap

editor occurance

Description: Count how many times something appears in the text.

Usage: ,editor occurance

editor replace

Description: Replace certain parts of the text.

Usage: ,editor replace

editor upper

Description: Convert the text to uppercase.

Usage: ,editor upper

editor shlex

Description: Shlex split a given string.

Usage: ,editor shlex

editor lower

Description: Convert the text to lowercase.

Usage: ,editor lower

editor bullet

Description: Bullet point a selection of items.

Usage: ,editor bullet

editor title

Description: Convert the text to titlecase.

Usage: ,editor title

editor snake

Description: Convert all spaces to underscores.

Usage: ,editor snake

editor permutate

Description: Generate permutations for given combinations of words/digits.

Usage: ,editor permutate
