NitroRole Help

Welcome new nitro boosters and/or give them a special role!

nitrorole

Description: Settings for NitroRole cog.

Usage: ,nitrorole

nitrorole unsetimage

Description: Unset image for new booster message.

Usage: ,nitrorole unsetimage

nitrorole setimage

Description: Set image for new booster message.

Usage: ,nitrorole setimage

nitrorole autoassignrole

Description: Set role that will be autoassigned after someone boosts server.

Leave empty to not assign any role.

Usage: ,nitrorole autoassignrole

nitrorole listmessages

Description: List new booster message templates.

Usage: ,nitrorole listmessages

nitrorole channel

Description: Set channel for new booster messages. Leave empty to disable.

Usage: ,nitrorole channel

nitrorole unassignonboostend

Description: Set if booster role should be unassigned when someone stops boosting server.

Leave empty to see current settings.

Usage: ,nitrorole unassignonboostend

nitrorole removemessage

Description: Remove new booster message.

Usage: ,nitrorole removemessage

nitrorole addmessage

Description: Add new booster message.

Those fields will get replaced automatically: $mention - Mention the user who boosted $username - The user's display name $server - The name of the server $count - The number of boosts server has (this isn't the same as amount of users that boost this server) $plural - Empty if count is 1. 's' otherwise

Note: New booster message can also have image. To set it, use [p]nitrorole setimage

Usage: ,nitrorole addmessage
