Blinder Help

Create custom roles that remove certain roles and hide channels from users.

blinder

Description: Group command for blinder.

Usage: ,blinder

blinder list

Description: List the roles currently managed by blinder.

Usage: ,blinder list

blinder forceenable

Description: Enable a blinder for a particular member.

Surround the member and role in quotes if their name requires a space.

Usage: ,blinder forceenable

blinder remove

Description: Stop a role from being managed by blinder.

Surround the role in quotes if its name requires a space.

Usage: ,blinder remove

blinder add

Description: Add a role to be managed by blinder.

Specify the role to be managed first. Specify any roles that should be removed when this role is applied after. Surround each role in quotes if its name requires a space.

Usage: ,blinder add

blinder edit

Description: Edit what roles are removed by a managed role.

Specify the role to be edited first. Specify any roles that should be removed when this role is applied after. Surround each role in quotes if its name requires a space.

Usage: ,blinder edit

blinder disable

Description: Disable a blinder for yourself.

Surround the role in quotes if its name requires a space.

Usage: ,blinder disable

blinder forcedisable

Description: Disable a blinder for a particular member.

Surround the member and role in quotes if their name requires a space.

Usage: ,blinder forcedisable

blinder enable

Description: Enable a blinder for yourself.

Surround the role in quotes if its name requires a space.

Usage: ,blinder enable

blinder toggleall

Description: Set every role in the guild to be removed when this role is applied.

Surround the role in quotes if its name requires a space.

Usage: ,blinder toggleall
