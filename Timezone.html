Timezone Help

Gets times across the world...

time

Description: Checks the time.

For the list of supported timezones, see here: https://en.wikipedia.org/wiki/List_of_tz_database_time_zones

Usage: ,time

time compare

Description: Compare your saved timezone with another user's timezone.

Usage: ,time compare

time me

Description: Sets your timezone. Usage: [p]time me Continent/City Using the command with no timezone will show your current timezone, if any.

Usage: ,time me

time version

Description: Show the cog version.

Usage: ,time version

time set

Description: Allows the bot owner to edit users' timezones. Use a user id for the user if they are not present in your server.

Usage: ,time set

time tz

Description: Gets the time in any timezone.

Usage: ,time tz

time user

Description: Shows the current time for the specified user.

Usage: ,time user

time iso

Description: Looks up ISO3166 country codes and gives you a supported timezone.

Usage: ,time iso
