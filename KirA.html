KirA Help

Keep It Relevant, Asshole!

kira

Description: Remind people to only post relevant links.

Usage: ,kira

kira timeout

Description: Set the timeout for questioning the sender. 0 will disable the questioning and deletes the message immediately.

Default is 10 seconds.

Usage: ,kira timeout

kira unwatch

Description: Remove a channel from the watchlist.

Usage: ,kira unwatch

kira domain

Description: Configure which domains to look out for.

This function doesn't do anything at the moment, but will be expanded later.

Usage: ,kira domain

kira watch

Description: Add a channel to be monitored.

Usage: ,kira watch

kira question

Description: Change the question the sender will be required to answer.

Usage: ,kira question
