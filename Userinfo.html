Userinfo Help

Replace original Red userinfo command with more details.

uinfoset

Description: Manage userinfo settings.

Usage: ,uinfoset

uinfoset clear

Description: Reset emojis to default.

Usage: ,uinfoset clear

uinfoset banner

Description: Toggle banner on userinfo.

Note: This causes a fetch request which can be a heavy operation.

Usage: ,uinfoset banner

userinfo

Description: Show userinfo with some more detail.

Usage: ,userinfo
