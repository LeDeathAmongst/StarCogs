GuildLock Help

Tools for managing guild joins and leaves.

guildlock

Description: View GuildLock settings

Usage: ,guildlock

guildlock leave

Description: Make the bot leave certain servers.

Leave Arguments - botfarms: leave servers with a bot ratio above the set percentage. - minmembers: leave servers with a member count below the set amount. - blacklist: leave any servers in the blacklist. - whitelist: leave any server not in the whitelist.

Usage: ,guildlock leave

guildlock whitelist

Description: Add/Remove a guild from the whitelist.

To remove a guild from the whitelist, specify it again.

Usage: ,guildlock whitelist

guildlock blacklist

Description: Add/Remove a guild from the blacklist.

To remove a guild from the blacklist, specify it again.

Usage: ,guildlock blacklist

guildlock channel

Description: Set the log channel for the bot

Usage: ,guildlock channel

guildlock limit

Description: Set the maximum amount of guilds the bot can be in.

Set to 0 to disable the guild limit.

Usage: ,guildlock limit

guildlock view

Description: View servers that fall under the auto-leave thresholds.

Arguments - botfarms: show servers with a bot ratio above the set percentage. - minmembers: show servers with a member count below the set amount. - blacklist: show any servers in the blacklist. - whitelist: show any server not in the whitelist.

Usage: ,guildlock view

guildlock minmembers

Description: Set the minimum number of members a server should have for the bot to stay in it.

Set to 0 to disable.

Usage: ,guildlock minmembers

guildlock botratio

Description: Set the the threshold percentage of bots-to-members for the bot to auto-leave.

Example If bot ratio is 60% and it joins a guild with 10 members (7 bots and 3 members) it will auto-leave since that ratio is 70%.

Set to 0 to disable.

Usage: ,guildlock botratio
