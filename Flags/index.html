Flags Help

Get flags from country names.

flag

Description: Get the flag for a country.

Either the country name or alpha 2 code can be provided.

Examples:

- ``[p]flag russia``
- ``[p]flag brazil``
- ``[p]flag dk``
- ``[p]flag se``

Usage: ,flag

flagemojis

Description: Get flag emojis for a list of countries.

Examples:

- ``[p]flagemojis qatar brazil mexico``
- ``[p]flagemojis "solomon islands" germany``

Usage: ,flagemojis

flags

Description: Get a list of all the flags and their alpha 2 codes.

Usage: ,flags
