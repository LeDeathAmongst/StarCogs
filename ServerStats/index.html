avatar

Description: Display a users avatar in chat

Usage: ,avatar

emoji

Description: Post a large size emojis in chat

Usage: ,emoji

botstats

Description: Display stats about the bot

Usage: ,botstats

channeledit

Description: Modify channel options

Usage: ,channeledit

channeledit nsfw

Description: Set whether or not a channel is NSFW

Usage: ,channeledit nsfw

channeledit bitrate

Description: Edit a voice channels bitrate

  • <channel> The voice channel you want to change.
  • <bitrate> The new bitrate between 8000 and 96000.

Usage: ,channeledit bitrate

channeledit topic

Description: Edit a channels topic

Usage: ,channeledit topic

channeledit userlimit

Description: Edit a voice channels user limit

  • <channel> The voice channel you want to change the limit on.
  • <limit> The limt on number of users between 0 and 99.

Usage: ,channeledit userlimit

channeledit permissions

Description: Edit channel read permissions for designated role

[channel] The channel you would like to edit. If no channel is provided the channel this command is run in will be used. [true_or_false] True or False to set the permission level. If this is not provided None will be used instead which signifies the default state of the permission. [roles_or_users...] the roles or users you want to edit this setting for.

<permission> Must be one of the following: - create_instant_invite - manage_channels - add_reactions - priority_speaker - stream - read_messages - send_messages - send_tts_messages - manage_messages - embed_links - attach_files - read_message_history - mention_everyone - external_emojis - connect - speak - mute_members - deafen_members - move_members - use_voice_activation - manage_roles - manage_webhooks - use_application_commands - request_to_speak - manage_threads - create_public_threads - create_private_threads - external_stickers - send_messages_in_threads - use_soundboard

Usage: ,channeledit permissions

channeledit position

Description: Edit a channels position

Usage: ,channeledit position

channeledit name

Description: Edit a channels name

Usage: ,channeledit name

channeledit sync

Description: Set whether or not to sync permissions with the channels Category

Usage: ,channeledit sync

pruneroles

Description: Perform various actions on users who haven't spoken in x days

Note: This will only check if a user has talked in the past x days whereas discords built in Prune checks online status

Usage: ,pruneroles

pruneroles list

Description: List the users who have not talked in x days.

  • <days> The days you want to search for.
  • [role] The role you want to check.

Usage: ,pruneroles list

pruneroles kick

Description: Kick users from the server who have been inactive for x days

  • <days> is the number of days since last seen talking on the server
  • [role] is the specified role you would like to kick defaults to everyone
  • [reinvite=True] True/False whether to try to send the user a message before kicking

Usage: ,pruneroles kick

pruneroles add

Description: Give roles to users who haven't spoken in x days

  • <days> is the number of days since last seen talking on the server
  • [new_roles...] The new roles to apply to a user who is inactive

Usage: ,pruneroles add

pruneroles remove

Description: Remove roles from users who haven't spoken in x days.

  • <days> is the number of days since last seen talking on the server.
  • [removed_roles...] the roles to remove from inactive users.

Usage: ,pruneroles remove

setguildjoin

Usage: ,setguildjoin

removeguildjoin

Description: Stop bots join/leave server messages

Usage: ,removeguildjoin

checkcheater

Description: Checks for possible cheaters abusing the global bank and server powers

Usage: ,checkcheater

whois

Description: Display servers a user shares with the bot

  • <user_id> The user you want to search for, ID's are preferred but some name lookup works.
  • Note: This will only show shared servers between you and the bot.

Usage: ,whois

topservers

Description: Lists servers by number of users and shows number of users

Usage: ,topservers

newservers

Description: Lists servers by when the bot was added to the server

Usage: ,newservers

guildedit

Description: Edit various guild settings

Usage: ,guildedit

guildedit afktimeout

Description: Change the servers AFK timeout

  • <timeout> must be a value of 60, 300, 900, 1800, or 3600.

Usage: ,guildedit afktimeout

guildedit name

Description: Change the server name

  • <name> The new name of the server.

Usage: ,guildedit name

guildedit systemchannel

Description: Change the system channel

This is the default discord welcome channel. - [channel] The channel you want to set as the system channel. - If not provided will be set to None.

Usage: ,guildedit systemchannel

guildedit verificationlevel

Description: Modify the guilds verification level

  • <level> must be one of:
  • none
  • low
  • medium
  • table flip
  • high
  • double table flip
  • extreme

Usage: ,guildedit verificationlevel

guildedit afkchannel

Description: Change the servers AFK voice channel

  • [channel] The channel you want to set as the system channel.
  • If not provided will be set to None.

Usage: ,guildedit afkchannel

topmembers

Description: Lists top members on the server by join date

  • [include_bots] whether or not to display bots or members. By default this will show everyone.
  • [guild] can be either the server ID or name.
  • Note: You must share the guild with the bot for this to work.

Usage: ,topmembers

listchannels

Description: Lists channels and their position and ID for a server

  • [guild] can be either the guild ID or name.
  • Note: You must share the guild with the bot for this to work.

Usage: ,listchannels

getguild

Description: Display info about servers the bot is on

  • [guild] can be either the guild ID or name.
  • Note: You must share the guild with the bot for this to work.

Usage: ,getguild

getguilds

Description: Display info about multiple servers

  • [guilds] can be multiple either the guild ID or name.
  • Note: You must share the guild with the bot for this to work.

Usage: ,getguilds

nummembers

Description: Display number of users on a server

  • [guild] can be either the guild ID or name.
  • Note: You must share the guild with the bot for this to work.

Usage: ,nummembers

getroles

Description: Displays all roles their ID and number of members in order of hierarchy

  • [guild] can be either the guild ID or name.
  • Note: You must share the guild with the bot for this to work.

Usage: ,getroles

getreactions

Description: Gets a list of all reactions from specified message and displays the user ID, Username, and Discriminator and the emoji name.

Usage: ,getreactions

serverstats

Description: Gets total messages on the server and displays each channel separately as well as the user who has posted the most in each channel

Note: This is a very slow function and may take some time to complete

Usage: ,serverstats

channelstats

Description: Gets total messages in a specific channel as well as the user who has posted the most in that channel

limit must be a number of messages to check, defaults to all messages Note: This can be a very slow function and may take some time to complete

Usage: ,channelstats

guildemojis

Description: Display all server emojis in a menu that can be scrolled through

id_emojis return the id of emojis. Default to False, set True if you want to see emojis ID's. - [guild] can be either the guild ID or name. - Note: You must share the guild with the bot for this to work.

Usage: ,guildemojis
