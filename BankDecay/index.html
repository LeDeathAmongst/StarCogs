BankDecay Help

Economy decay!

Periodically reduces users' red currency based on inactivity, encouraging engagement. Server admins can configure decay parameters, view settings, and manually trigger decay cycles. User activity is tracked via messages and reactions.

bankdecay

Description: Setup economy credit decay for your server

Usage: ,bankdecay

bankdecay ignorerole

Description: Add/Remove a role from the ignore list

Users with an ignored role will not have their balance decay

Usage: ,bankdecay ignorerole

bankdecay initialize

Description: Initialize the server and add every member to the config.

Arguments - as_expired: (t/f) if True, initialize users as already expired

Usage: ,bankdecay initialize

bankdecay toggle

Description: Toggle the bank decay feature on or off.

Usage: ,bankdecay toggle

bankdecay bulkrempercent

Description: Remove a percentage from all member balances.

Accidentally refunded too many credits with bulkaddpercent? Bulk remove from every user's balance in the server based on a percentage of their current balance.

Usage: ,bankdecay bulkrempercent

bankdecay decaynow

Description: Run a decay cycle on this server right now

Usage: ,bankdecay decaynow

bankdecay logchannel

Description: Set the log channel, each time the decay cycle runs this will be updated

Usage: ,bankdecay logchannel

bankdecay getexpired

Description: Get a list of users who are currently expired and how much they will lose if decayed

Usage: ,bankdecay getexpired

bankdecay setdays

Description: Set the number of inactive days before decay starts.

Usage: ,bankdecay setdays

bankdecay bulkaddpercent

Description: Add a percentage to all member balances.

Accidentally decayed too many credits? Bulk add to every user's balance in the server based on a percentage of their current balance.

Usage: ,bankdecay bulkaddpercent

bankdecay seen

Description: Check when a user was last active (if at all)

Usage: ,bankdecay seen

bankdecay view

Description: View Bank Decay Settings

Usage: ,bankdecay view

bankdecay cleanup

Description: Remove users from the config that are no longer in the server or have no balance

Usage: ,bankdecay cleanup

bankdecay setpercent

Description: Set the percentage of decay that occurs after the inactive period.

Example If decay is 5%, then after the set days of inactivity they will lose 5% of their balance every day.

Usage: ,bankdecay setpercent

bankdecay resettotal

Description: Reset the total amount decayed to zero.

Usage: ,bankdecay resettotal
