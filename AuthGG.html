AuthGG Help

authgg

Description: Control your users from auth.gg straight from Discord

Usage: ,authgg

authgg resethwid

Description: Reset a user's HWID lock on auth.gg for the specified API key name.

The API key name must be the friendly name provided by [p]authgg keys add.

Usage: ,authgg resethwid

authgg changepw

Description: Set a user's password to the specified input for the specified API key name.

The API key name must be the friendly name provided by [p]authgg keys add.

Usage: ,authgg changepw

authgg keys

Description: Manage API keys for auth.gg

Usage: ,authgg keys

authgg keys remove

Description: Remove an auth.gg API key via its friendly name

Usage: ,authgg keys remove

authgg keys clear

Description: Clears all auth.gg API keys

Usage: ,authgg keys clear

authgg keys list

Description: Lists registered auth.gg API keys by their friendly name

Usage: ,authgg keys list

authgg keys add

Description: Register an auth.gg API key under a friendly name

Usage: ,authgg keys add

authgg roles

Description: Control what roles have access to reseting a user's HWID

Usage: ,authgg roles

authgg roles add

Description: Add a role to the whitelist for resetting user's HWID

Usage: ,authgg roles add

authgg roles remove

Description: Remove a role from the whitelist for resetting user's HWID

Usage: ,authgg roles remove

authgg roles clear

Description: Remove all currently whitelisted roles

Usage: ,authgg roles clear

authgg roles list

Description: View all currently whitelisted roles

Usage: ,authgg roles list
