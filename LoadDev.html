LoadDev Help

Allow live loading of Dev commands from Core

devset

Description: Settings for automatically loading/unloading cores dev using this cog

Usage: ,devset

devset autoload

Description: Whether or not to automatically load dev when this cog is loaded.

Note: This does not affect the CLI flag --dev

Usage: ,devset autoload

devset replacemock

Description: Set an automatic replacement for [p]mock when auto loading dev.

Usage: ,devset replacemock

loaddev

Description: Dynamically loads dev cog

[replace_mock] the replacement command name for [p]mock If nothing is provided this will not replace [p]mock.

Usage: ,loaddev

unloaddev

Description: Unload Dev

Usage: ,unloaddev
