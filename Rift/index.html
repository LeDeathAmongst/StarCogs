Rift Help

Communicate with other servers/channels.

send

Description: Send a message to the specified destinations.

Editing or deleting the message you send will still forward to the bot's reposts, as in normal rifts.

Usage: ,send

rift

Description: Communicate with other channels through Red.

Usage: ,rift

rift notify

Description: Toggle whether the bot notifies the destination of an open rift.

The notification is only disabled for bot owners, and will still notify channels the bot owner doesn't have direct access to.

Usage: ,rift notify

rift info

Description: Provides info about rifts opened in the specified scope.

Usage: ,rift info

rift close

Description: Closes all rifts that lead to this channel.

Usage: ,rift close

rift close guild

Description: Closes all rifts that lead to this server.

Usage: ,rift close guild

rift link

Description: Links this channel to the specified destination(s).

Anything anyone says in this channel will be forwarded. All replies will be relayed back here.

Usage: ,rift link

rift web

Description: Opens up all possible connections between this channel and the specified rifts.

See the helptext of [p]rift link for more info.

Usage: ,rift web

rift open

Description: Opens a rift to the specified destination(s).

Only your messages will be forwarded to the specified destinations, and all replies will be sent back to you.

Usage: ,rift open

rift blocklist

Description: Configures blocklists.

Blocklisted destinations cannot have rifts opened to them.

Usage: ,rift blocklist

rift blocklist server

Description: Blocklists the current server.

All channels and members in a server are considered blocklisted if the server is blocklisted. Members can still be reached if they are in another, non-blocklisted server.

Usage: ,rift blocklist server

rift blocklist channel

Description: Blocklists the current channel or the specified channel.

Can also blocklist DM channels.

Usage: ,rift blocklist channel
