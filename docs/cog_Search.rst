Search
======

# ,wiki
Search wikipedia for provided search term, and return the first result.<br/>
 - Usage: `,wiki <term>`
Extended Arg Info
> ### *term: str
> ```
> A single word, if not using slash and multiple words are necessary use a quote e.g "Hello world".
> ```
