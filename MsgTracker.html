MsgTracker Help

Tracks how many messages people send.

Idea prompted by Esports Discord, specifically Frostyy#1313 (526672641701183509)

mtset

Description: Message Count Settings.

Members are allowed to deny permission from being tracked for privacy reasons unless the Bot Owner sets differently.

Usage: ,mtset

mtset userlist

Description: Displays users who are being ignored from tracking.

Thank you Core Red

Usage: ,mtset userlist

mtset ignore

Description: Ignore users / channels from message tracking.

Usage: ,mtset ignore

mtset ignore user

Description: This blocks the user(s) from being tracked.

To remove someone from the list, use the command again with the same user.

Members will have access to a command that lets them do it themselves if they so choose.

Usage: ,mtset ignore user

mtset ignore channel

Description: This will ignore/unignore a channel from message tracking.

If a channel is added to the list, type it again to remove it.

Usage: ,mtset ignore channel

mtset ignore mods

Description: Ignores/Unignores staff of a server.

This will do it automatically based on what the previous setting is.

Usage: ,mtset ignore mods

mtset enable

Description: Enables / Disables system.

This will do it automatically based on what the previous setting is.

Usage: ,mtset enable

mtset channellist

Description: Displays channels that are being ignored from tracking.

Thank you Core Red

Usage: ,mtset channellist

mtset admin

Description: Administration commands.

Allows you to nuke someone's history from record or adjust their count.

Usage: ,mtset admin

mtset admin nukeconfig

Description: [BOT OWNER] Nukes all recorded counters.

This will nuke every messages that's been counted. Do so at your own risk.

Usage: ,mtset admin nukeconfig

mtset admin counter

Description: Change a member's recorded message count.

Usage: ,mtset admin counter

mtset admin blockignore

Description: [BOT OWNER] Disables [p]trackignore command for users

Usage: ,mtset admin blockignore

mtset admin deletecount

Description: Will delete the member's entire record.

Will ask for confirmation if you want to do this.

Usage: ,mtset admin deletecount

mtset admin cache

Description: [BOT OWNER] Check cached messages counter for this server.

Usage: ,mtset admin cache

mtset admin resetguild

Description: Will delete Guild's recorded members.

Will ask for confirmation if you want to do this.

Usage: ,mtset admin resetguild

mtset admin updateconfig

Description: [BOT OWNER] Updates to config manually

Usage: ,mtset admin updateconfig

mtset showsettings

Description: Display guild specific settings.

This will not display blocked users, please use [p]mtset ignorelist. This will not display ignored channels, please use [p]mtset channellist.

Usage: ,mtset showsettings

trackignore

Description: Ignore yourself from message tracking.

If you want to be tracked for your message count, you will need to contact the staff of the current server.

Usage: ,trackignore

msgtop10

Description: Displays the top 10 members in the leaderboard

Usage: ,msgtop10

messages

Description: Displays how many messages the user has.

If user is none, defaults to author. Thank you The Discord Historian#0666!

Usage: ,messages

msgleaderboard

Description: Message Tracker Leaderboard, displays all the members.

Credit to Core Economy

Usage: ,msgleaderboard
