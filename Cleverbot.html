Cleverbot Help

Cleverbot rewritten for V3 from
https://github.com/Twentysix26/26-Cogs/tree/master/cleverbot

cleverbot

Description: Talk with cleverbot

Usage: ,cleverbot

cleverbotset

Description: Settings for cleverbot

Usage: ,cleverbotset

cleverbotset channel

Description: Toggles channel for automatic replies

do [p]cleverbot channel after a channel is set to disable.

Usage: ,cleverbotset channel

cleverbotset allowlist

Description: Whitelist settings for cleverbot

Usage: ,cleverbotset allowlist

cleverbotset allowlist add

Description: Add a channel, user, or role to cleverbots whitelist

[channel_user_role...] is the channel, user or role to whitelist (You can supply more than one of any at a time)

Usage: ,cleverbotset allowlist add

cleverbotset allowlist info

Description: Show what's currently in cleverbots whitelist

Usage: ,cleverbotset allowlist info

cleverbotset allowlist remove

Description: Remove a channel, user, or role from cleverbots whitelist

[channel_user_role...] is the channel, user or role to remove from the whitelist (You can supply more than one of any at a time)

Usage: ,cleverbotset allowlist remove

cleverbotset guildtweaks

Description: Set the response tweaks from cleverbot

<tweak1> a number from 0-100 varies replies from sensible to wacky <tweak2> a number from 0-100 varies replies from shy to talkative <tweak3> a number from 0-100 varies replies from self-centered to attentive

Setting any tweak to -1 will default to the bots global tweaks.

Usage: ,cleverbotset guildtweaks

cleverbotset mention

Description: Toggles mention on reply

Usage: ,cleverbotset mention

cleverbotset apikey

Description: Sets token to be used with cleverbot.com You can get it from https://www.cleverbot.com/api/ Use this command in direct message to keep your token secret

Usage: ,cleverbotset apikey

cleverbotset tweakinfo

Description: Show the current cleverbot tweaks in this server

Usage: ,cleverbotset tweakinfo

cleverbotset ioapikey

Description: Sets token to be used with cleverbot.io You can get it from https://www.cleverbot.io/ Use this command in direct message to keep your token secret

Usage: ,cleverbotset ioapikey

cleverbotset tweaks

Description: Set the response tweaks from cleverbot

<tweak1> a number from 0-100 varies replies from sensible to wacky <tweak2> a number from 0-100 varies replies from shy to talkative <tweak3> a number from 0-100 varies replies from self-centered to attentive

Usage: ,cleverbotset tweaks

cleverbotset toggle

Description: Toggles reply when the bot is mentioned

Usage: ,cleverbotset toggle

cleverbotset reply

Description: Toggles reply messages

Note: This is only available for Red 3.4.6 and discord.py 1.6.0

Usage: ,cleverbotset reply

cleverbotset dm

Description: Toggles reply in DM

Usage: ,cleverbotset dm

cleverbotset blocklist

Description: Blacklist settings for cleverbot

Usage: ,cleverbotset blocklist

cleverbotset blocklist info

Description: Show what's currently in cleverbots blacklist

Usage: ,cleverbotset blocklist info

cleverbotset blocklist remove

Description: Remove a channel, user, or role from cleverbots blacklist

[channel_user_role...] is the channel, user or role to remove from the blacklist (You can supply more than one of any at a time)

Usage: ,cleverbotset blocklist remove

cleverbotset blocklist add

Description: Add a channel, user, or role to cleverbots blacklist

[channel_user_role...] is the channel, user or role to blacklist (You can supply more than one of any at a time)

Usage: ,cleverbotset blocklist add
