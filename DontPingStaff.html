DontPingStaff Help

Punish users for pinging staff.

dps

Description: Dont ping staff.

Usage: ,dps

dps settings

Description: Show the current settings.

Usage: ,dps settings

dps staffrole

Description: Command for managing the staff role.

Usage: ,dps staffrole

dps staffrole list

Description: List the staff roles.

Usage: ,dps staffrole list

dps staffrole add

Description: Add a role to the staff role.

Usage: ,dps staffrole add

dps staffrole remove

Description: Remove a role from the staff role.

Usage: ,dps staffrole remove

dps scope

Description: Set the scope of the module.

Scope - guild - Enable DPS server-wide by passing guild. - category - Enable DPS for a specific category by passing its category ID. - channel - Enable DPS for a specific channel by passing its channel ID.

Note: You can specify multiple categories and channels, separated by a space. Running the command again will override the previous configuration.

Example - To enable DPS for specific channels, categories, or a mix of both: dps scope 123456789 123456789 123456789

Usage: ,dps scope
