ModRoles Help

Allow moderators to assign configured roles to users.

assignrole

Description: Assign a role to a member.

NOTE: The role is case sensitive!

Usage: ,assignrole

unassignrole

Description: Unassign a role from a member.

NOTE: The role is case sensitive!

Usage: ,unassignrole

modroles

Description: Settings for assignable roles.

Usage: ,modroles

modroles targets

Description: Settings about allowed targets.

Usage: ,modroles targets

modroles targets allowbots

Description: Allow to assign roles to bots with [p]assignrole

Leave empty to check current settings.

Usage: ,modroles targets allowbots

modroles targets toprole

Description: Enable/disable top role check.

When enabled, this will only allow user to assign roles to users with lower top role than theirs.

Leave empty to check current settings.

Usage: ,modroles targets toprole

modroles add

Description: Add assignable role.

Usage: ,modroles add

modroles remove

Description: Remove assignable role.

Usage: ,modroles remove

modroles list

Description: List assignable roles.

Usage: ,modroles list
