Bumper Help

A cog for bumping your server to other servers.

bumpset

Description: Group command to set bump configuration.

Usage: ,bumpset

bumpset embed_color

Description: Set the embed color.

Usage: ,bumpset embed_color

bumpset invite

Description: Set the invite link.

Usage: ,bumpset invite

bumpset channel

Description: Set the bump channel.

Usage: ,bumpset channel

bumpset description

Description: Set the server description (max 500 characters).

Usage: ,bumpset description

bumpowner

Description: Owner-only bump configuration.

Usage: ,bumpowner

bumpowner remove_trusted

Description: Remove a trusted user.

Usage: ,bumpowner remove_trusted

bumpowner generate_code

Description: Generate a premium code.

Usage: ,bumpowner generate_code

bumpowner bump_log_channel

Description: Set the channel where bump logs are sent.

Usage: ,bumpowner bump_log_channel

bumpowner add_trusted

Description: Add a trusted user to accept or deny reports.

Usage: ,bumpowner add_trusted

bumpowner blacklist

Description: Blacklist a server by ID.

Usage: ,bumpowner blacklist

bumpowner unblacklist

Description: Unblacklist a server by ID.

Usage: ,bumpowner unblacklist

bumpowner report_channel

Description: Set the channel where bump reports are sent.

Usage: ,bumpowner report_channel

bumpowner config_log_channel

Description: Set the channel where configuration logs are sent.

Usage: ,bumpowner config_log_channel

redeem

Description: Redeem a premium code.

Usage: ,redeem

bumper

Description: Send the bump message to all servers with a configured bump channel.

Usage: ,bumper

bumprep

Description: Group command for handling bump reports.

Usage: ,bumprep

bumprep deny

Description: Deny a reported bump.

Usage: ,bumprep deny

bumprep accept

Description: Accept a reported bump.

Usage: ,bumprep accept
