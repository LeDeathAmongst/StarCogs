Session Help

Shows how many commands are invoked in a session (resets on reboot)

session

Description: Session group commands.

Usage: ,session

session type

Description: Define the session game type.

Type list: 0 = Playing 2 = Listening 3 = Watching 5 = Competing

Usage: ,session type

session delay

Description: Sets interval of random status switch.

Must be 20 or superior.

Usage: ,session delay

session status

Description: Define the session presence status.

Status list: 0 = Online 1 = Idle 2 = DND 3 = Invisible

Usage: ,session status
