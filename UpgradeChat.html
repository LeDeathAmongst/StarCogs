UpgradeChat Help

Upgrade.Chat API integration for buying economy credits directly instead of roles

https://upgrade.chat/

upgradechat

Description: Base command for cog settings

Usage: ,upgradechat

upgradechat delproduct

Description: Delete an Upgrade.Chat product by UUID

Usage: ,upgradechat delproduct

upgradechat tokens

Description: Set your Upgrade.Chat api tokens By using this feature it is assumed that you are already familiar with Upgrade.Chat

  1. Create your api keys here: https://upgrade.chat/developers

  2. Copy your client ID and Client Secret

  3. Run this command with your credentials

Enjoy!

Usage: ,upgradechat tokens

upgradechat purchases

Description: View user purchase history

Usage: ,upgradechat purchases

upgradechat message

Description: Set the message the bot sends when a user claims a purchase

Valid placeholders: {mention} - mention the user {username} - the users discord name {displayname} - the users nickname(if they have one) {uid} - the users Discord ID {guild} - guild name {creditsname} - name of the currency in your guild {amount} - the amount of credits the user has claimed

set to default to use the default message

Usage: ,upgradechat message

upgradechat logchannel

Description: Set log channel for claims

Usage: ,upgradechat logchannel

upgradechat view

Description: View your current products

Usage: ,upgradechat view

upgradechat addproduct

Description: Add an Upgrade.Chat product by UUID

This can be any type of product, either subscription or one-time purchase. Users will be accredited based on amount spend * conversion ratio. Transactions can only be claimed once.

Usage: ,upgradechat addproduct

upgradechat ratio

Description: Set the worth of 1 unit of real currency to economy credits

for example, if credit_worth is 100, then $1 = 100 Credits

Usage: ,upgradechat ratio

claim

Description: Claim your Upgrade.Chat purchases!

Usage: ,claim
