Grafana Help

Grafana graphs in your Discord!

graph

Description: Render an image of a selected panel of [botname] metrics.

Usage: ,graph

graph list

Description: List all panels that can be used with [p]graph command.

Usage: ,graph list

graph set

Description: Setup grafana cog.

Usage: ,graph set

graph set panels

Description: Setup graphs on dashboard.

Usage: ,graph set panels

graph set panels add

Description: Add certain graph to list manually.

Usage: ,graph set panels add

graph set panels remove

Description: Remove certain graph from list.

Usage: ,graph set panels remove

graph set panels import

Description: Automatically import all graphs from dashboard, overwriting already saved.

Usage: ,graph set panels import

graph set showsettings

Description: Show current settings.

Usage: ,graph set showsettings

graph set dashboard

Description: Set dashboard id.

This command needs id from URL. Example: http://localhost:3000/d/AbCdEf0G/dashboard ^ here ^

Usage: ,graph set dashboard

graph set url

Description: Setup url of your Grafana instance.

Default: http://localhost:3000

Usage: ,graph set url
