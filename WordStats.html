WordStats Help

Tracks commonly used words.

wordstats

Description: Prints the most commonly used words.

Use the optional parameter "member_or_guild" to see the stats of a member or guild. Use the optional parameter "amount_or_word" to change the number of words that are displayed or to check the stats of a specific word.

Usage: ,wordstats

wordstats global

Description: Prints the most commonly used words across all guilds.

Use the optional parameter "amount_or_word" to change the number of words that are displayed or to check the stats of a specific word.

Usage: ,wordstats global

topchatters

Description: Prints the members who have said the most words.

Use the optional parameter "guild" to see the topchatters in a specific guild. Use the optional parameter "word" to see the topchatters of a specific word. Use the optional parameter "amount" to change the number of members that are displayed.

Usage: ,topchatters

topchatters global

Description: Prints the members who have said the most words across all guilds.

Use the optional parameter "word" to see the topchatters of a specific word. Use the optional parameter "amount" to change the number of members that are displayed.

Usage: ,topchatters global

topratio

Description: Prints the members with the highest "word to all words" ratio.

Use the parameter "word" to set the word to compare. Use the optional parameter "guild" to see the ratio in a specific guild. Use the optional parameter "amount" to change the number of members that are displayed. Use the optional parameter "min_total" to change the minimum number of words a user needs to have said to be shown.

Usage: ,topratio

topratio global

Description: Prints the members with the highest "word to all words" ratio in all guilds.

Use the parameter "word" to set the word to compare. Use the optional parameter "amount" to change the number of members that are displayed. Use the optional parameter "min_total" to change the minimum number of words a user needs to have said to be shown.

Usage: ,topratio global

wordstatsset

Description: Config options for wordstats.

Usage: ,wordstatsset

wordstatsset minlength

Description: Set the minimum length a word has to be in order to be displayed.

Shorter words will still be included in numerical counts, they will only be hidden from list displays. Set to 0 to disable. Defaults to 0. This value is server specific.

Usage: ,wordstatsset minlength

wordstatsset deleteall

Description: Delete all wordstats data.

This removes all existing data, creating a blank state. This cannot be undone.

Usage: ,wordstatsset deleteall

wordstatsset user

Description: Set if wordstats should record stats for you.

Defaults to True. This value is user specific.

Usage: ,wordstatsset user

wordstatsset forgetme

Description: Make wordstats forget all data about you.

This is equivalent to [p]mydata forgetme, but only for wordstats. This cannot be undone.

Usage: ,wordstatsset forgetme

wordstatsset server

Description: Set if wordstats should record stats for this server.

Defaults to True. This value is server specific.

Usage: ,wordstatsset server

wordstatsset stopwords

Description: Set if stopwords should be included in outputs.

Stopwords are common words such as "a", "it" and "the". Stopwords will still be included in numerical counts, they will only be hidden from list displays. Defaults to True. This value is server specific.

Usage: ,wordstatsset stopwords

wordstatsset channel

Description: Set if wordstats should record stats for this channel.

Defaults to True. This value is channel specific.

Usage: ,wordstatsset channel
