ListMaker Help

Make lists to store data.

listmaker

Description: Group command for ListMaker.

Usage: ,listmaker

listmaker show

Description: View the data of a list.

Wrap the list name in quotes if it requires spaces. Use the parameter show_index to enable list indexes.

Example: [p]listmaker show "My friends"

Usage: ,listmaker show

listmaker header

Description: Set the text to be used as a header for a list.

Wrap anything that requires spaces in quotes. The text {date} will be replaced by the date the list was last updated.

Usage: ,listmaker header

listmaker remove

Description: Remove a row of data from a list.

Specify the list name first and the row number after. Wrap the list name in quotes if it requires spaces.

Example: [p]listmaker remove "My friends" 3

Usage: ,listmaker remove

listmaker create

Description: Create a new list.

Specify the list name first and the name of each column after. Wrap any names that require spaces in quotes.

Example: [p]listmaker create "My friends" name age

Usage: ,listmaker create

listmaker list

Description: List the existing listmaker lists.

Also shows the user id of the author of each list. Use the parameter show_all to see all lists.

Usage: ,listmaker list

listmaker role

Description: Toggle if a role_id should be allowed to edit a list.

Wrap the list name in quotes if it requires spaces.

Usage: ,listmaker role

listmaker delete

Description: Delete a list.

This deletes the list and all of its contents permanently. Wrap the list name in quotes if it requires spaces.

Example: [p]listmaker delete "My friends"

Usage: ,listmaker delete

listmaker add

Description: Add a row of data to a list.

Specify the list name first and the data for each column after. Wrap anything that requires spaces in quotes. Multiple rows can be added at once by continuing to specify rows.

Example: [p]listmaker add "My friends" "Robert Smith" 26

Usage: ,listmaker add

listmaker sort

Description: Set the list to be sorted by a particular column.

Wrap anything that requires spaces in quotes. Use the parameter reverse to sort the other way.

Usage: ,listmaker sort
