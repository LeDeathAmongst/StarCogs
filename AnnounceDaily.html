    AnnounceDaily Help

    Send daily announcements

    announcedaily

    Description: Base command for managing AnnounceDaily settings

    Do [p]help annd <subcommand> for more details

    Usage: ,announcedaily

    announcedaily addimg

    Description: Add an image to the pool of announcement images

    You must attach an image while executing this command

    Usage: ,announcedaily addimg

    announcedaily delimg

    Description: Remove an image from the announcement pool

    Does not delete the file from the disk, so you may have to clean it up occasionally

    Usage: ,announcedaily delimg

    announcedaily listmsg

    Description: List all registered announcement messages

    Usage: ,announcedaily listmsg

    announcedaily setchannel

    Description: Set the announcement channel for this server

    Don't pass a channel to clear this server of receiving announcements

    Usage: ,announcedaily setchannel

    announcedaily settime

    Description: Set the daily announcement time

    It will first announce at the time you provided, then it will repeat every 24 hours

    Usage: ,announcedaily settime

    announcedaily listimg

    Description: List all registered announcement images

    Usage: ,announcedaily listimg

    announcedaily addmsg

    Description: Add a message to the pool of announcement messages

    Usage: ,announcedaily addmsg

    announcedaily delmsg

    Description: Remove a message from the announcement pool

    Must provide the index of the message, which can be found by using [p]annd listmsg

    Usage: ,announcedaily delmsg

    runannounce

    Description: Manually run the daily announcement

    Usage: ,runannounce
