Spoilerer Help

spoiler

Description: Spoilers the attachments provided with the message for you.

The optional [message] argument will be posted along with the spoilered attachments. The message itself will remain as-is, without any spoilering.

Usage: ,spoiler

spoiler button

Description: Enable or disable the spoiler button for this guild.

The spoiler button adds ðŸ”³ as a reaction to any attachments sent by members that are on mobile or that are invisible. Clicking this button acts as if they used the [p]spoiler command.

Usage: ,spoiler button

spoiler message

Description: Spoilers the specified message's attachments by deleting it and re-posting it.

Pass True to spoiler_content to also spoiler the message text, e.g.: [p]spoiler message True 1053802538056548392

If message is not specified, then this will default to the message you reply to when using the command.

Usage: ,spoiler message
