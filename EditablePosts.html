EditablePosts Help

editable_posts

Description: Group command for creating posts you can edit later.

Usage: ,editable_posts

editable_posts create

Description: Creates a new editable post in a given channel. Use the edit dubcommand to change its text.

Usage: ,editable_posts create

editable_posts list

Description: Lists all editable posts on this server.

Usage: ,editable_posts list

editable_posts remove

Description: Deletes an editable post.

Usage: ,editable_posts remove

editable_posts title

Description: Changes the title of an editable post.

Usage: ,editable_posts title

editable_posts edit

Description: Edits an editable post with new text.

Usage: ,editable_posts edit
