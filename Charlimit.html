Charlimit Help

Limit the amount of characters / lines per channel

charlimit

Description: Manage the character limits

Usage: ,charlimit

charlimit message

Description: Allows/Disallows the bot to send a message to users if they exceed the limit.

Default is False, to allow type [p]charlimit message True othwerise no messages will be sent.

Usage: ,charlimit message

charlimit list

Description: The list of the channels in category and character limit

Usage: ,charlimit list

charlimit set

Description: Sets the character and line limimts

Usage: ,charlimit set

charlimit set char

Description: Sets the channel's character limit

If you put 0 for a channel, it'll remove the character limit. example: [p]charlimit set char #general 0

Usage: ,charlimit set char

charlimit set lines

Description: Sets the channel's line limit

If you put 0 for a channel, it'll remove the line limit. example: [p]charlimit set lines #general 0

Usage: ,charlimit set lines
