Xenon Help

Cog for saving and loading server templates.

savet

Description: Saves the current server's structure as a template.

Usage: ,savet

addtrusted

Description: Adds a trusted user who can run the savet and loadt commands.

Usage: ,addtrusted

remtrusted

Description: Removes a trusted user.

Usage: ,remtrusted

loadt

Description: Loads a template and applies it to the current server.

This command will delete all existing channels and roles on the server and recreate them based on the specified template.

Parameters

template_id : str The ID of the template to load.

Usage: ,loadt

listt

Description: Lists all saved templates.

Usage: ,listt
